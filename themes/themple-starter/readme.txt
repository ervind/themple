== Description ==
Themple Starter is a starter theme for the Themple Framework. Themple Framework is a code package for WordPress theme developers with a goal to make their lives easier.

Key features:
* Theme Options for use in templates, post content, CSS (LESS) and JS files
* Page Builder
* Responsive Grid Layout
* Custom Color schemes
* Built in TGM plugin activation & Font Awesome
* Pre-defined styles for the most common HTML and WP elements
* Speed optimized front end
* Detailed documentation
* GPL v2.0 license. Use it freely and create something cool

For more information about Themple please go to https://a-idea.studio/themple-framework/

== Installation ==

1. In your admin panel, go to Appearance > Themes and click the 'Add New' button.
2. Click on Upload Theme and upload your downloaded copy of the Themple Starter ZIP file.
3. Click on the 'Activate' button to use your new theme right away.
4. Go to https://a-idea.studio/themple-framework/ for a guide on how to customize this theme.
5. Navigate to Appearance > Theme Options in your admin panel where you can customize it.
6. For adding some extra functionality, install the Themple Helper plugin (which is bundled with your theme).
